# Un generateur de mot de passe.
Un générateur de mots de passe moderne, sécurisé et extensible. Ce projet d'apprentissage du développement web, de la programmation et des bonnes pratiques Git/GitHub.

## Objectifs du projet

1. Créer un générateur de mots de passe robuste

2. Appliquer des pratiques modernes en sécurité et développement

3. Structurer un projet comme dans une vraie entreprise tech

## Fonctionnalités prévues

1. Génération de mots de passe aléatoires

2. Longueur personnalisée

3. Choix des types de caractères (majuscules, minuscules, symboles, chiffres)

4. Option d'exclusion de caractères ambigus

5. Génération multi-mots de passe

6. Interface simple (CLI ou Web selon la version)

7. Tests automatisés

8. Bonnes pratiques de sécurité
